A Linguagem Básica da Web: HTML

Postado por Laura Jones em 21 de Junho 2027, Segunda-feira

Todos os websites modernos e aplicações da web são construídos usando três tecnologias fundamentais: HTML, CSS e JavaScript. Essas são as linguagens da web.

Nesse post, vamos focar no HTML. Nós aprenderemos o que é HTML e porquê você deveria aprendê-lo.

O Que É HTML?

HTML é uma abraviação para HyperText Markup Language, ou Linguagem de Marcação de Hipertexto. É uma linguagem de marcação que os desenvolvedores web utilizam para estruturar e descrever o conteúdo de uma página da web (não é uma linguagem de programação).

HTML consiste de elementos que descrevem tipos diferentes de conteúdo: parágrafos, links, títulos, imagens, vídeos, etc. Navegadores da web entendem HTML e renderizam código HTML como websites.

Em HTML, cada elemento é composto de 3 partes:

  1. Um marcador de abertura
  2. Um marcador de fechamento
  3. O elemento em si

Você pode aprender mais em MDN Web Docs.

Por que você deveria aprender HTML?

Existem inúmeros razões para aprender a linguagem fundamental da web. Abaixo você encontra 5 delas:

  • Para ser capaz de usar a linguagem fundamental de desenvolvimento web
  • Para criar websites lindos autorais ao invés de depender de ferramentas como Wordpress ou Wix
  • Para criar aplicações web
  • Para impressionar seus amigos
  • Para se divertir 😃
